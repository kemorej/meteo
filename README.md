
# Météo

Une application web simple et efficace pour afficher les prévisions météo en temps réel pour plusieurs villes en France. Ce projet utilise l'API OpenWeatherMap pour récupérer et afficher les données météo actuelles, les prévisions horaires, et les alertes météo.

---

## 📌 Fonctionnalités

- **Recherche par ville** : Saisissez le nom d'une ville pour obtenir sa météo actuelle.
- **Affichage des villes prédéfinies** : Météo actuelle pour Marseille, Lille, Brest, Strasbourg et Paris.
- **Prévisions horaires** : Affichage des prévisions pour les 6 prochaines heures.
- **Prévisions pour demain** : Météo prévue à 9h, 12h et 17h le lendemain.
- **Alertes météo** : Affichage des alertes en cours (si disponibles).
- **Carte interactive** : Localisation de la ville sélectionnée via OpenStreetMap.
- **Responsive design** : Adapté aux mobiles, tablettes et ordinateurs.

---

## 🛠 Technologies Utilisées

- **PHP** : Langage principal pour la logique backend.
- **OpenWeatherMap API** : Récupération des données météo.
- **OpenStreetMap** : Affichage de la carte de localisation.
- **HTML/CSS** : Structure et style de l'interface utilisateur.
- **JavaScript** : Interactions basiques (redirection, etc.).

---

## 🚀 Installation

### Prérequis

- Un serveur web (Apache, Nginx, etc.) avec support PHP.
- Une clé API OpenWeatherMap (gratuite pour un usage limité).
